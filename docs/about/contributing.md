# Contributing

We welcome contributions to ts-stack!

## Ways to Contribute

- **Report bugs** — Open an issue with a minimal reproduction
- **Suggest features** — Discuss before starting major work
- **Submit PRs** — Fork, branch, commit, push, create PR
- **Improve docs** — Help clarify or expand documentation
- **Add test vectors** — Contribute conformance vectors

## Development

1. Clone the repo: `git clone https://github.com/bsv-blockchain/ts-stack`
2. Install dependencies: `pnpm install`
3. Run tests: `pnpm test`
4. Make your changes
5. Submit a PR

## Code Style

- TypeScript with strict mode
- 2 spaces for indentation
- Meaningful variable names
- Docstring comments for public APIs

## Testing

All changes require tests. Run:

```bash
pnpm test
```

## Reporting Issues

Please include:
- TypeScript version
- Node.js version
- Minimal reproduction
- Expected vs actual behavior

## License

## Questions?

Open an issue or reach out to the maintainers.
