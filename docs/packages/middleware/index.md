# Middleware Domain

Express.js middleware for adding authentication and payment gating to web services. Enables Bitcoin signature-based authentication and HTTP 402 payment requirements.

## Packages

- **@bsv/auth-express-middleware** — Express middleware for BRC-31 mutual authentication
- **@bsv/payment-express-middleware** — Express middleware for HTTP 402 payment gating
- **@bsv/402-pay** — Client-side HTTP 402 payment handler
