# @bsv/wallet-toolbox

BRC-100 wallet client library for integrating with BSV wallets.

## Features

- Connect to BRC-100 wallets
- Request signatures and encryption
- Manage wallet keys
- Support for local and remote wallets

## Installation

```bash
npm install @bsv/wallet-toolbox
```
