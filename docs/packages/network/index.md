# Network

Broadcast transactions and query BSV nodes through a typed client.

## Packages in this Domain

- [@bsv/teranode-listener](./ts-p2p.md) — Connect to Teranode and listen for transactions

## What You Can Do

- Broadcast signed transactions
- Query UTXOs
- Listen for new transactions
- Retrieve transaction history

## When to Use

Use network packages when you need to:

- Broadcast transactions to the chain
- Query account balance
- Monitor for incoming transactions
