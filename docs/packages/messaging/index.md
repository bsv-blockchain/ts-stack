# Messaging Domain

Real-time messaging, authentication, and identity protocols for BSV. Provides authenticated WebSockets, message box overlay service, and Paymail protocol implementation.

## Packages

- **@bsv/message-box-client** — Client for sending/receiving authenticated messages via MessageBox overlay
- **@bsv/authsocket** — Server-side BRC-31 authenticated WebSocket
- **@bsv/authsocket-client** — Client-side BRC-31 authenticated WebSocket
- **@bsv/paymail** — Paymail protocol (capability discovery, PKI, P2P payments, profiles)
